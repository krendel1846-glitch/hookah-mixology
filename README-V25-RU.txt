Hookah Mixology Engine Pro — v25 UI + scoring polish

Что изменено:
1. Русифицированы бейджи совместимости и концепций:
   - Body forward → Тело вперёд
   - Balanced → Баланс
   - Dessert rounded → Десертное округление
   - Contrast built → Контрастная сборка
   - Commercial safe → Коммерчески безопасно

2. Улучшена оценка качества:
   - 90–94%: отличный микс
   - 80–89%: хороший рабочий
   - 70–79%: условно хороший
   - 55–69%: спорный / рискованный
   - ниже 55%: слабый
   История ротации больше не должна обнулять оценку, а 100% теперь не ставится почти всем рабочим вариантам.

3. Улучшен визуал забивки:
   - секторная схема увеличена и лучше центрирована;
   - слоистая схема стала больше похожа на чашу;
   - подписи вкусов и процентов сохранены прямо в визуале.

4. Уточнён вывод архитектуры:
   - добавлена строка Связка / контраст;
   - роли bridge, rounder, contrast, cooler выводятся понятнее;
   - холод по-прежнему считается внутри общей процентовки.
